####
RIPWHINE - an easy-to-use single purpose ripper and encoder
####

Requires ``python-musicbrainz2``

Not configurable, you can set the destination dir in the UI.

A one-day hack job.

Usage
-----

* Insert CD
* i for identify
* r for rip
